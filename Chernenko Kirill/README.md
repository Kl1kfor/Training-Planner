## Описание программы
Training Planner - это простое desktop-приложение для ведения личного дневника тренировок. Программа позволяет добавлять, удалять и фильтровать тренировки, а также сохранять данные в JSON файл.

## Возможности программы:
1. Добавление тренировок с указанием даты, типа и длительности
2. Просмотр всех тренировок в виде таблицы
3. Фильтрация тренировок по типу (точное совпадение)
4. Фильтрация тренировок по дате (точное совпадение)
5. Удаление тренировок из коллекции
6. Сохранение данных в JSON файл
7. Загрузка данных из JSON файла при запуске
8. Валидация вводимых данных
9. Автоматический подсчёт статистики (общее количество, общая и средняя длительность)

## Проверка корректности ввода:
- **Дата:** 2026-04-28
- **Длительность:** должна быть положительным числом (целым или дробным)
